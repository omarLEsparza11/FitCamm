Sobre el proyecto

¿Qué es FitCam?

FitCam es una aplicación diseñada para ayudarte a mejorar tu postura al momento de realizar ejercicios físicos. Utiliza tecnología de análisis visual para detectar errores y ayudarte a entrenar correctamente.

Objetivo

Reducir lesiones, mejorar el rendimiento físico y brindar una experiencia de entrenamiento más inteligente.

Tecnología

Este proyecto combina visión por computadora, análisis en tiempo real y desarrollo web para crear una herramienta innovadora.

FitCam busca mejorar la postura usando tecnología y análisis visual.
